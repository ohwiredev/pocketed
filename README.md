# Pocketed 📁✨

**Your AI-Powered Video Library.**

Pocketed is a sleek, modern web application designed to help you curate your
personal video collection. Save any video from **TikTok**, **Instagram**, or
**YouTube** and find them instantly using intelligent AI-powered search and
automated tagging.

## 🚀 Features

- **Multi-Platform Support**: Save videos from TikTok, Instagram, and YouTube
  effortlessly.
- **AI-Powered Search**: Don't just search by title search by content, context,
  and mood.
- **Auto Tagging**: Let our AI categorize your videos so you don't have to.
- **Instant Retrieval**: A lightning fast interface built with React 19 and
  Vite.
- **Premium Design**: A beautiful, responsive UI crafted with Tailwind CSS 4.

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (Latest LTS recommended)
- [pnpm](https://pnpm.io/) (Recommended) or npm/yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/pocketed.git
   cd pocketed
   ```

2. **Install dependencies:**
   ```bash
   pnpm install
   ```

3. **Start the development server:**
   ```bash
   pnpm dev
   ```

4. **Build for production:**
   ```bash
   pnpm build
   ```
